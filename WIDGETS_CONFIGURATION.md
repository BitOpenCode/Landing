# Конфигурация виджетов для iPhone анимации

## Что такое виджеты?

Виджеты - это анимированные блоки с изображениями, которые разлетаются вокруг iPhone при клике на него.

## Структура виджета

Каждый виджет состоит из:
1. **HTML элемент** - `<img>` с классом `widgets` и уникальным `id`
2. **Изображение** - путь к изображению в атрибуте `src`
3. **Анимация** - настройки в массиве `animations` (позиция, масштаб, скорость)

## Как добавить свои изображения

### Шаг 1: Разместите изображения
Поместите ваши изображения в папку `public/images/`:
- `widget-1.png`
- `widget-2.png`
- и т.д.

### Шаг 2: Обновите пути в компоненте
В файле `src/components/IPhoneAnimation.jsx` найдите блоки виджетов и замените `src`:

```jsx
<img id="widget-1" className="widgets" src="/images/widget-1.png" alt="Widget 1" />
<img id="widget-2" className="widgets" src="/images/widget-2.png" alt="Widget 2" />
```

### Шаг 3: Настройте анимацию (опционально)
В массиве `animations` вы можете настроить для каждого виджета:

- **x, y** - позиция, куда виджет разлетается (в пикселях)
- **scale** - масштаб виджета (0.9 = меньше, 1.1 = больше)
- **duration** - длительность анимации (в секундах)
- **ease** - тип анимации ("power4.out", "power2.out" и т.д.)

Пример:
```javascript
{ 
  selector: "#widget-1",  // ID виджета
  duration: 3,            // 3 секунды
  scale: 0.9,             // Уменьшить до 90%
  x: 500,                 // Сдвинуть вправо на 500px
  y: 100,                 // Сдвинуть вниз на 100px
  ease: "power4.out"      // Плавная анимация
}
```

## Текущие виджеты

Сейчас настроено 11 виджетов:
- `widget-1` до `widget-11`

Вы можете:
- Удалить ненужные виджеты (удалите `<img>` и соответствующую запись в `animations`)
- Добавить новые виджеты (добавьте `<img>` и запись в `animations`)
- Изменить количество виджетов

## Размеры изображений

Рекомендуемые размеры для виджетов:
- Ширина: 200-400px
- Высота: 200-400px
- Формат: PNG, SVG (с прозрачностью)

## Примеры использования

### Минимальная конфигурация (3 виджета):
```jsx
<img id="widget-1" className="widgets" src="/images/feature-1.png" alt="Feature 1" />
<img id="widget-2" className="widgets" src="/images/feature-2.png" alt="Feature 2" />
<img id="widget-3" className="widgets" src="/images/feature-3.png" alt="Feature 3" />
```

И в массиве `animations` оставьте только первые 3 записи.
