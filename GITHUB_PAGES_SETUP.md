# Настройка GitHub Pages

## Шаги для активации GitHub Pages:

1. **Перейдите в настройки репозитория:**
   - Откройте https://github.com/BitOpenCode/Landing
   - Нажмите на вкладку **Settings** (вверху справа)

2. **Активируйте GitHub Pages:**
   - В левом меню найдите раздел **Pages**
   - В разделе **Source** выберите:
     - **Source**: `GitHub Actions` (не "Deploy from a branch")
   - Сохраните изменения

3. **Проверьте статус деплоя:**
   - Перейдите во вкладку **Actions** в репозитории
   - Убедитесь, что workflow "Deploy to GitHub Pages" запустился и успешно выполнился
   - Если есть ошибки, проверьте логи

4. **Дождитесь завершения деплоя:**
   - Первый деплой может занять 2-5 минут
   - После успешного деплоя сайт будет доступен по адресу:
     - `https://bitopencode.github.io/Landing/`

## Если сайт все еще не работает:

1. Проверьте, что в настройках репозитория включен GitHub Pages
2. Убедитесь, что workflow успешно выполнился (зеленая галочка в Actions)
3. Подождите несколько минут после первого деплоя
4. Очистите кэш браузера и попробуйте снова

## Альтернативный вариант (если GitHub Actions не работает):

Если GitHub Actions не работает, можно использовать ветку `gh-pages`:

```bash
npm install --save-dev gh-pages
```

Добавить в `package.json`:
```json
"scripts": {
  "deploy": "npm run build && gh-pages -d dist"
}
```

Затем запустить:
```bash
npm run deploy
```

Но рекомендуется использовать GitHub Actions, так как он уже настроен.
